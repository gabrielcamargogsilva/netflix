# Projeto Netflix Clone (Estudo)

Este repositório contém uma página web inspirada na interface de seleção de perfis da Netflix. O objetivo principal é reforçar conceitos de HTML semântico, CSS responsivo e boas práticas de acessibilidade.

## 🚀 Sobre
- Tipo: página de clonagem visual
- Uso: fins educacionais
- Inspiração: tela de seleção de perfil da Netflix

## 🧩 Tech Stack
- HTML5
- CSS3

## 🛠️ Funcionalidades implementadas
- Estrutura semântica com `<main>`, `<section>`, `<nav>`, `<ul>`, `<li>`, `<figure>`, `<figcaption>`
- Ícone de favicon customizado (tipo Netflix)
- Cards de perfil com hover e transição suave
- Layout responsivo (desktop / tablet / celular)

## 📁 Estrutura
- `index.html` - página principal
- `style.css` - estilos visuais
- `assets/` - imagens dos perfis e ícones

## 🔍 Observações
Trata-se de uma cópia para aprendizado, sem finalidade comercial ou uso oficial da marca Netflix. O projeto serve como base para treinar layout, semântica e responsividade.

## ▶️ Como executar
1. Abrir `index.html` no navegador.
2. Ou rodar com servidor local (ex: Live Server no VSCode).
